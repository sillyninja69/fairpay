## Environment Setup

Create a `.env.local` file and set:

```env
NEXT_PUBLIC_PROGRAM_ID=your_program_id
NEXT_PUBLIC_SOLANA_NETWORK=devnet
GEMINI_API_KEY=your_gemini_api_key
GEMINI_MODEL=gemini-1.5-flash
RESOLVER_SECRET_KEY=[1,2,3,...]
```

Notes:
- `GEMINI_API_KEY` is used by the dispute resolver endpoint.
- `GOOGLE_API_KEY` is also accepted as a fallback key name.
- `RESOLVER_SECRET_KEY` must be a JSON array format private key for the resolver wallet.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
